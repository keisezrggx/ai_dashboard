Langkah Peningkatan Limit

  1. Buka halaman PayLater
  2. Klik "Limit Tersedia"
  3. Di "Pusat Peningkatan Limit", pilih item otorisasi: Verifikasi Laporan Pajak, Isi Kartu Bank, Info Perusahaan, Email Perusahaan, SIM, NPWP, Kontak Darurat
  4. Lengkapi pengisian data

⚠ Persetujuan limit = operasi sistem, tidak bisa diintervensi manual

PUTARAN 3

⑧B — Limit Cukup Tapi Tidak Bisa Digunakan — Cek Tagihan

Pertanyaan: "Apakah saat ini Anda memiliki tagihan yang sedang berjalan di platform/aplikasi lain? Termasuk tagihan di Akulaku?"

1
Tidak Ada Tagihan & Tidak Pernah Terlambat
Kode 3.15
→ Skor Kredit / Penilaian Risiko Sistem
klik untuk detail ▾

Skrip CS

Masalah disebabkan oleh hasil skor kredit atau penilaian risiko. Riwayat keterlambatan di aplikasi lain dapat memengaruhi keberhasilan. Persetujuan limit = operasi sistem, tidak dapat diintervensi manual.

2
Ada Tagihan di Platform Akulaku
Kode 3.17
→ Lunasi Cicilan, Coba 90 Hari Lagi
klik untuk detail ▾

Skrip CS

Tagihan berjalan di Akulaku dapat memengaruhi keberhasilan transaksi. Selesaikan cicilan terlebih dahulu, coba bertransaksi kembali setelah 90 hari.

3
Ada Tagihan di Platform Eksternal
Kode 3.16
→ Lunasi Tagihan Eksternal, 90 Hari, No New Bills
klik untuk detail ▾

Skrip CS

Tagihan di aplikasi lain dapat memengaruhi belanja PayLater. Selesaikan tagihan/cicilan yang ada, coba kembali setelah 90 hari. Selama periode tersebut, jangan tambah tagihan baru di Akulaku maupun aplikasi eksternal.
